Virtual Environment (venv)

Why and how to use venv on Windows, macOS, and Linux.

What is venv?

Isolates Python packages per project, avoiding conflicts with global packages.

Why use it?
  • Dependency isolation
  • Reproducible environments
  • Safe experimentation
Create & Activate
# create
python -m venv venv

# activate (Windows)
venv\Scripts\activate

# activate (macOS/Linux)
source venv/bin/activate

# deactivate
deactivate
