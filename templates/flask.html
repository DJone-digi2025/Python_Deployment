Flask Framework

Lightweight Python web framework. Covers architecture, request-response, routing, and Jinja2 templates.

What is Flask?

A micro web framework that is simple and extensible.

Why Flask?
  • Minimal core
  • Easy routing & templates
  • Great for learning and small projects
Architecture

Client → Flask App → WSGI Server → Response. Uses Werkzeug & Jinja2 internally.

Routing & Jinja2

Routes map URL paths to functions. Jinja2 templates allow dynamic HTML.

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

if __name__ == "__main__":
    app.run(debug=True)
