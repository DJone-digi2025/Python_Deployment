Interpreter vs Compiler

Differences explained clearly with examples and hybrid Python model.

Interpreter

Executes source code line by line at runtime.

▶ Details
Good for interactive development and debugging. Provides precise error locations.
Compiler

Translates full source code into machine code or bytecode before execution.

▶ Details
Produces faster executable code. Can distribute without source. Errors found during compilation.
Python

CPython compiles to bytecode (.pyc) and interprets it using a VM — hybrid model.

Alternative implementations: PyPy (JIT), Jython, IronPython.
