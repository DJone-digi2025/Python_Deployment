About This Project

A classroom-focused, interactive platform to teach Python fundamentals clearly and effectively.

Key Features

  • Clean, responsive UI built with Tailwind CSS
  • Animated cards and hover-based learning interactions
  • Maze-style click-to-reveal concepts for active classrooms
  • Flask backend with Jinja templating
  • Modular architecture using base templates

Who This Is For

This project is ideal for:

  • Students learning Python for the first time
  • Teachers conducting live or online classes
  • Coaching centers and bootcamps
  • Self-learners who prefer visual explanations

How to Run Locally

Follow these steps to run the project on your system: 

python -m venv venv

# Activate virtual environment
venv\Scripts\activate        # Windows
source venv/bin/activate     # macOS / Linux

# Install dependencies
pip install flask

# Run application
python app.py

# Open in browser
http://127.0.0.1:5000
