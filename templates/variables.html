Variables & Datatypes

Learn about variables, datatypes, input/output, and operators in Python.

Variables

What is a variable?

A named reference to a value in memory. Assignment uses =.

▶ Show example
name = "Alice"
age = 21
pi = 3.14159
print(name, age)
Datatypes

Common datatypes

  • int — integers
  • float — floating point numbers
  • str — text
  • bool — True / False
▶ Examples
a = 10      # int
b = 2.5     # float
s = "hello" # str
f = True    # bool
Input & Output

input() and print()

Use input() to read from user and print() to display output.

▶ Demo
name = input("Enter name: ")
print("Hello,", name)
Operators

Arithmetic & Comparison

  • + add, - subtract, * multiply, / divide
  • ==, <, >, !=
▶ Examples
5 + 3   # 8
5 / 2   # 2.5
5 // 2  # 2 (floor)
5 % 2   # 1 (remainder)
